Turing Wars

Players write programs which compete in deathmatch duels inside a virtual machine.

A hackathon collaboration of 4 people which was further improved over the course of a few months.

The Game UI is built on React with TypeScript. The script parser is written in TypeScript.
The execution engine is written in Scala and compiled for the web with Scala.js.
